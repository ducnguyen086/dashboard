# 🏛 TRẠM TRUNG CHUYỂN DỮ LIỆU DASHBOARD

Hệ thống Dashboard Mirae Asset của bạn được thiết kế theo mô hình "Cùng nhau xây dựng". Dưới đây là hướng dẫn để bạn cập nhật dữ liệu hàng ngày một cách chuyên nghiệp nhất:

## 📅 Quy trình 17h hàng ngày:

1.  **Chuẩn bị:** Tải file **PDF** bản tin hoặc **Ảnh chụp** bảng số liệu của Mirae Asset về.
2.  **Import:** Copy (hoặc kéo thả) file đó vào thư mục này (`C:\Users\Admin\.gemini\antigravity\scratch\finance_dashboard\import_data`).
3.  **Kích hoạt AI:**
    *   Mở khung chat **Antigravity** (nơi bạn đang nói chuyện với tôi).
    *   Nhắn: **"Cập nhật Dashboard từ folder import đi"**.
4.  **Hoàn tất:** Tôi sẽ tự động "đọc" dữ liệu trong file bạn cung cấp, trích xuất chính xác mọi thông số và ghi thẳng vào mã nguồn Dashboard.
5.  **Tận hưởng:** Bạn chỉ cần nhấn **F5** trên trình duyệt để thấy báo cáo mới nhất.

## ⚙️ Ghi chú:
- Bạn có thể xóa các file cũ trong thư mục này sau khi đã cập nhật thành công.
- Nếu bạn có **Hình ảnh (Charts)**, hãy để nguyên tên file, tôi sẽ tự động liên kết chúng vào Dashboard cho bạn.
